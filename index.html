Infrastructure & Security Projects

Data Center Server Migration & Infrastructure Enhancement

Nov 2021 - Mar 2022

Associated with: Chesmi Consolidated (pvt) Ltd.

Overview: Led a critical infrastructure overhaul to replace aging legacy hardware (Dell T110 ii) with a modern, virtualized environment. The goal was to ensure high availability, redundancy, and centralized management.

  • Virtualization: Migrated 3 bare-metal servers to 2 Dell R440 hosts running VMware ESXi 7.
  • OS Modernization: Upgraded MS Dynamics NAV (Server 2019 → 2022) and Active Directory (Server 2008 R2 → 2016).
  • Physical Upgrades: Renovated server room with new racks, redundant UPS systems, and dual-WAN load balancing.
  • Security: Deployed SonicWall firewalls with LDAP-integrated SSL VPN for secure remote access.
  • Backup Strategy: Configured Nakivo Backup & Replication on a QSAN NAS for disaster recovery.
